# Sabrina Olliphant Portfolio

Static portfolio site for Sabrina Olliphant, focused on full-stack product engineering, AI-assisted mobile apps, Python data projects, and clear technical documentation.

## Local Preview

Open `index.html` directly in a browser.

## Deploying To GitHub Pages

1. Create a public GitHub repository named `portfolio`.
2. Push this repo to `git@github.com:msvelouria-hue/portfolio.git`.
3. In GitHub, open Settings > Pages.
4. Set source to `Deploy from a branch`.
5. Select `main` and `/root`.

Expected public URL:

```text
https://msvelouria-hue.github.io/portfolio/
```
